# Interact

Basic account Smart contract with metamask interaction.

## Setup

- install dependencies

```shell
npm install
```

- Compile smart contract

```shell
npx hardhat compile
```

- Run tests

```shell
npx hardhat test
```

- Run local hardhat node

```shell
npx hardhat node
```

- Deploy smart contract

```shell
npx hardhat run scripts/deploy.js --network localhost
```

- Start the app

```shell
npm run start
```

Technologies

- Hardhat
- EtherJS
- React
